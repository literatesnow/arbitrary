# Arbitrary

Draws various objects to an arbitrary height.

* A Tree
* An X

## Setting Up

[Vagrant](https://www.vagrantup.com/) is required.

```bash
  me$ vagrant up
  me$ vagrant ssh
  vagrant$ cd /vagrant
  vagrant$ bundle install
```

## Running in Vagrant

```bash
  vagrant$ cd /vagrant
  vagrant$ bundle exec bin/arbitrary
```

## Tests in Vagrant

```bash
  vagrant$ cd /vagrant
  vagrant$ bundle exec rspec
```
